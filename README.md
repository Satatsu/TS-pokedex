# TS-pokedex
### <span style="color: #3498db;">Technologies : </span>
- The website is coded using:
  - React (TSX)
  - TypeScript
  - Scss
## <span style="color: #5d7bc7">_Getting Started</span>
```bash
cd TS-pokedex
```
```bash
npm install
```
```bash
npm start
```
